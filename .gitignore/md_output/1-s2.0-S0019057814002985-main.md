# Markdown Summary

This paper introduces two adaptive Proportional-Integral (PI) control schemes aimed at regulating web tension in Roll-to-Roll (R2R) manufacturing systems. These schemes are vital for ensuring the quality of continuous materials processed via R2R systems, such as printing and coating.

The first scheme utilizes a model reference approach where controller gains are tuned to match an actual closed-loop system with a specified reference model. The second adopts an indirect adaptive control method that leverages relay feedback technique for automatic initialization of PI gains. Notably, both schemes simplify the tuning process by automating parameter adjustment in response to changing conditions.

Experimentations were conducted on a large experimental R2R machine simulating various industrial practices, involving different polymer webs and operational settings. This study demonstrates enhanced adaptive performance over traditional fixed-gain PI controllers used in industry, with significant implications for automation and efficiency improvements in R2R manufacturing environments.

# Roll-to-Roll (R2R) Manufacturing Control

This paper introduces Model Predictive Path Integral (MPPI) control applied to Roll-to-Roll (R2R) manufacturing systems, highlighting improved precision and scalability. The MPPI framework solves real-time control challenges encountered in R2R processes like tension regulation, which historically relied on proportional-integral (PI) controllers or more complex techniques such as model predictive control (MPC). Unlike MPC, MPPI efficiently manages non-differentiable cost functions and adapts to stochastic dynamics without solving computationally intensive optimization problems at each step. The Monte-Carlo sampling method, enhanced with GPU acceleration, allows for scalable and real-time optimal control computation in high-dimensional, nonlinear systems characteristic of R2R manufacturing environments. A case study in the paper illustrates significant improvements in tension regulation using MPPI over traditional MPC methods.
